CAN YOU GO 8-0?

Sorteie uma seleção e uma copa, escale um jogador para cada posição e descubra se o seu time consegue ser campeão invicto.

3. Escale o time: Selecione jogadores daquela equipe e monte o seu "Dream Team".
4. Simule o campeonato: Quando terminar, simule os jogos para ver se você consegue ganhar 8 partidas seguidas sem perder!
