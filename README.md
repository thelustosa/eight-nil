# 8-0 Draft / Sorteador

Um aplicativo web interativo para sortear seleções e copas, além de permitir o "draft" de jogadores para compor o seu esquadrão ideal com um visual de cartas colecionáveis.

**Jogue agora mesmo no seu navegador:**
https://thelustosa.github.io/eight-nil/

## Recursos Principais
- Sorteio aleatório de Seleção e Copa.
- Interface de seleção de formação tática e estilo de jogo.
- Draft de jogadores baseado na seleção/copa sorteada.
- Lista final do esquadrão com interface estilo "card".

## Como usar
Basta abrir o `index.html` no seu navegador favorito. Não é necessário nenhum servidor ou build para rodar.

## Tecnologias
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
